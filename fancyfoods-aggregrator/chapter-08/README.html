Chapter 8

This source code demonstrates various techniques for testing OSGi projects.

The build scripts and instructions assume you have already run the chapter 2 and chapter 3 samples.

Pax-exam itests

  1. Set an ARIES_ASSEMBLY environment variable pointing to the blog-assembly folder of the Apache Aries sandbox. For example, 
    export ARIES_ASSEMBLY=~/aries/samples/blog/blog-assembly
  2. Run mvn install in the chapter-08/fancyfoods.department.cheese.itest directory to build and run the pax-exam tests.
  3. The maven build should run tests and complete successfully
